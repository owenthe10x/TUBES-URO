# Tugas Besar URO 2023 Kelompok 1
## Overview
Program merupakan program dimana user berperan sebagai kapten yang bertugas menghancurkan kapal musuh sebanyak mungkin.

## Cara menjalankan
Compile dengan menjalankan:
```g++ -o main main.cpp```

Run dengan menjalankan:
```./main```
